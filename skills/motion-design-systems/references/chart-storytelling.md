# Chart Storytelling Reference

Use this when charts, curves, timelines, adoption gaps, value gaps, maturity stages, or score trajectories carry the story.

## Narrative Order

1. Establish the chart frame with subtle axes and labels.
2. Reveal starting point.
3. Draw primary curve progressively.
4. Draw comparison curve at its own pace.
5. Reveal points only when the curve reaches each point.
6. Reveal labels near their lines.
7. Reveal gap/area annotations only after the data relationship is visible.

## Labeling Rules

- Put labels next to the curve or region they explain.
- Avoid legends when the user wants immediate comprehension.
- Use black or brand-neutral text for secondary curves unless the brand system says otherwise.
- Make gap labels readable but do not let them cover the data path.

## Failure Modes

- Points pre-marked before the line is drawn.
- Gap fill appears before both boundaries exist.
- Legend appears after the user already needs the meaning.
- Labels appear too late to support narration.
- A carryover chart in the next scene lacks a new label or theme change.

## Proof Frames

Always extract frames at:

- Axis established.
- First point reached.
- Main gap label appears.
- Curve labels appear.
- Final chart state.
- Next-scene carryover start.
